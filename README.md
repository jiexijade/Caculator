## 安卓多功能计算器项目说明

本项目将在 Android Studio 中实现一个面向手机端的多功能计算器应用。计算器是日常生活中常用工具，本应用将集成多种类型的计算和转换功能，覆盖基础运算、科学计算、贷款计算、汇率换算和单位换算等场景，并在 Android 设备上完成运行演示。

### 1. 基本计算器（必选，对应页面：`activity_calculator.xml`）

- **功能要求**
  - 支持十进制基本运算：
    - 加法、减法、乘法、除法
    - 支持带括号的运算表达式（如 \((3+5)\times 2\)）
    - 平方运算（如 \(x^2\)）
    - 开平方运算（如 \(\sqrt{x}\)）
  - 支持十六进制基本运算：
    - 至少支持十六进制的加、减、乘、除运算
    - 提供十进制与十六进制之间的显示或输入方式（可根据 UI 设计决定切换方式）
- **界面要求**
  - 结合 `app/src/main/res/layout/activity_calculator.xml` 布局页面进行功能实现与交互设计。

### 2. 科学计算器 / 复数计算（可选，对应页面：`activity_complex_calculator.xml`）

- **功能要求（可选扩展）**
  - 在基本运算基础上，支持复数的基本运算：
    - 复数加、减、乘、除
    - 支持类似第 1 点中的表达式形式（如带括号运算）
- **界面要求**
  - 结合 `app/src/main/res/layout/activity_complex_calculator.xml` 布局页面完成复数输入、结果显示和运算符操作。

### 3. 房贷 / 车贷计算器（必选，对应页面：`activity_loan_calculator.xml`）

- **功能要求**
  - 支持房贷和车贷的常见贷款计算功能，要求计算精度高。
  - 至少实现以下两种还款方式：
    - 等额本金计算
    - 等额本息计算
  - 支持根据专业贷款计算方法进行公式设计和优化（可参考金融学中标准房贷/车贷计算公式）。
  - 输入内容包括但不限于：贷款总额、贷款年限/期数、年利率或月利率、还款方式选择等。
- **界面要求**
  - 结合 `app/src/main/res/layout/activity_loan_calculator.xml` 布局页面设计输入控件、结果展示区域及计算按钮。

### 4. 汇率计算器（必选，对应页面：`activity_currency_calculator.xml`）

- **功能要求**
  - 实现多币种之间的汇率转换功能（例如人民币与美元、欧元等之间的换算）。
  - 不使用网络接口或实时网络请求，采用当前预设汇率数据进行计算。
  - 支持双向换算（如从币种 A 转为币种 B，或从币种 B 转为币种 A）。
  - 可根据需要设计汇率更新方式（如在界面中提供手动修改汇率的入口，是否要求“实时更新”可按课程/实验要求确定）。
- **界面要求**
  - 结合 `app/src/main/res/layout/activity_currency_calculator.xml` 页面，实现币种选择、金额输入与结果显示。

### 5. 单位换算功能（必选，对应页面：`activity_length_converter.xml`、`activity_speed_converter.xml`、`activity_weight_converter.xml`）

- **功能要求**
  - 实现常用单位换算功能，至少包含以下几类：
    - 长度单位换算（如米、千米、厘米、英寸等）
    - 重量单位换算（如千克、克、吨、磅等）
    - 速度单位换算（如米/秒、千米/小时等）
    - 数据单位换算（如字节、KB、MB、GB 等，可根据需要设计对应页面或整合到现有页面）
  - 保证换算公式正确，换算结果精度合理。
- **界面要求**
  - 长度换算：结合 `app/src/main/res/layout/activity_length_converter.xml` 布局实现。
  - 速度换算：结合 `app/src/main/res/layout/activity_speed_converter.xml` 布局实现。
  - 重量换算：结合 `app/src/main/res/layout/activity_weight_converter.xml` 布局实现。
  - 如需扩展数据单位换算，可新建或复用相应布局页面。

### 6. 自定义扩展功能（可选）

- **可选扩展示例**
  - 历史记录功能：记录用户最近的计算表达式和结果，支持点击回显或再次计算。
  - 主题切换：支持明/暗色主题或多套配色方案。
  - 个税计算、小费计算、油耗计算等其他生活实用计算功能。
  - 自定义常用汇率或常用单位组合，方便快速选择。
- **要求**
  - 至少保证基础功能稳定，在此基础上根据时间和兴趣选择 1–2 个扩展功能进行实现。

### 7. Android 平台演示要求（必选）

- **运行环境**
  - 使用 Android Studio 进行开发与调试。
  - 目标平台为 Android 手机设备（可使用真机或模拟器）。
- **演示要求**
  - 所有上述必选功能（基本计算器、贷款计算器、汇率计算、单位换算）必须在 Android 系统上可正常运行并完成示范。
  - 应用应提供清晰的导航或菜单，让用户可以方便地在不同计算模块之间切换。
  - 界面布局美观、交互友好，适配常见手机屏幕尺寸。
